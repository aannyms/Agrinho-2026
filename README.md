# 🌱 Agrinho 2026 – Sustentabilidade no Agro

Este projeto foi desenvolvido com base no tema do programa Agrinho 2026, que propõe uma reflexão sobre a importância do agronegócio aliado à sustentabilidade e à responsabilidade social. A ideia central do site é mostrar que é possível produzir alimentos, desenvolver a economia e, ao mesmo tempo, preservar o meio ambiente, criando um equilíbrio entre o campo e a sociedade.

O desenvolvimento do projeto começou a partir da análise do tema proposto, buscando criar algo que fosse simples de entender, visualmente agradável e que transmitisse uma mensagem clara. A proposta não foi apenas fazer um site bonito, mas sim construir uma experiência que chamasse a atenção do usuário e o fizesse refletir sobre a importância do agro sustentável.

A estrutura do site foi organizada de forma estratégica. Inicialmente, foi criada a base em HTML, definindo todas as seções como o cabeçalho, o menu de navegação, a área principal com imagem de destaque, os textos explicativos, os cards informativos, a parte de práticas sustentáveis e o contato. Cada parte foi pensada para guiar o usuário de forma natural ao longo da página, facilitando a leitura e o entendimento.

Após a estrutura, foi feita toda a estilização utilizando CSS. Foram escolhidas cores em tons de verde para representar a natureza, o campo e a sustentabilidade. Além disso, foram aplicados efeitos modernos, como sombras, bordas arredondadas e transições suaves, deixando o site mais profissional e agradável visualmente. Também foi trabalhada a responsividade, garantindo que o site funcione bem tanto em celulares quanto em computadores.

Na parte de interatividade, foi utilizado JavaScript para tornar o site mais dinâmico. Foram implementadas funcionalidades como o scroll suave entre as seções, animações ao rolar a página e um botão de contato com feedback visual ao usuário. Essas interações ajudam a melhorar a experiência de quem acessa o site, deixando tudo mais moderno e envolvente.

O projeto foi desenvolvido de forma gradual, passando por etapas de planejamento, construção, testes e ajustes. Após a finalização, o site foi publicado utilizando o GitHub Pages, permitindo que ele ficasse disponível online para acesso público.

Mais do que um simples trabalho, este projeto representa a aplicação prática de conhecimentos de desenvolvimento web, além de reforçar uma mensagem importante: o futuro depende do equilíbrio entre produção e preservação. O agronegócio tem um papel fundamental na sociedade, e quando aliado a práticas sustentáveis, pode contribuir diretamente para um mundo melhor.

Por fim, este projeto também abre espaço para melhorias futuras, como a implementação de um formulário de contato real, novas animações, maior acessibilidade e até integração com outras tecnologias. Isso mostra que o desenvolvimento é um processo contínuo, sempre podendo evoluir.
